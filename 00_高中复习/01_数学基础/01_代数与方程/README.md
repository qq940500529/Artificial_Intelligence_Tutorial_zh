# 代数与方程

> **所属路径**：`00_高中复习/01_数学基础/01_代数与方程`
> **包含知识点**：5 个

---

## 概述

代数与方程是整个数学体系的基石。在人工智能领域，几乎所有的模型训练都可以归结为"解方程"——找到一组参数，使得某个目标函数取到最优值。理解方程的本质、掌握方程的求解技巧，是后续学习线性代数、微积分和最优化理论的必要前提。

本主题从最基础的一元二次方程出发，逐步引入不等式与绝对值的分析方法，通过方程组与代数变形帮助你建立"多变量协同求解"的思维方式，再通过幂运算与根式掌握指数和根号的运算规则，最后学习不等式证明的核心技巧。五个知识点环环相扣，为你在后续阶段处理更复杂的数学问题打下坚实基础。

## 学习建议

1. **学习顺序**：建议按 01 → 02 → 03 → 04 → 05 的顺序学习。一元二次方程中的求解思路和判别式概念会在后续不等式分析中反复使用，方程组求解是对单个方程求解能力的自然扩展，幂运算与根式为指数与对数打下基础，不等式证明综合运用前面所学的代数技巧。
2. **时间分配**：每个知识点预计 30–60 分钟，整个主题约 4–5 小时。
3. **常见困难**：许多学习者容易在"配方法"和"代数变形"上感到困惑，建议遇到卡顿时回到具体数字例子，先算出结果再理解一般规律。
