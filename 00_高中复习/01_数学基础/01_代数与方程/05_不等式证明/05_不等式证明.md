## 正文讲解

### 1. 为什么不等式需要"证明"

在前面的课程里，我们学会了如何"求解"不等式——给定一个不等式，找出使它成立的变量取值范围。但数学中还有另一类问题：已知某个条件，需要论证某个不等关系**一定成立**。这就是 **不等式证明（Inequality Proof）** 。

来看一个日常场景：你和朋友一起去超市，每人买了若干水果。你可能凭直觉觉得"买的总量的平均值，一定不会小于每人购买量的几何平均"。但直觉并不足以让别人信服——我们需要一个严密的论证过程，确保这个结论在**所有可能的情况**下都成立。这就是"证明"的价值。

在人工智能领域，不等式证明的思想随处可见。例如：

- 训练神经网络时，我们需要证明损失函数有一个**下界**，这样优化算法才知道有可能收敛到最优解。
- **[正则化（Regularization）](../../../../01_基础能力/02_数学基础/04_最优化/)** 的理论分析中，需要证明添加惩罚项后目标函数满足特定的不等式约束。
- 信息论中的吉布斯不等式（Gibbs' inequality）证明了交叉熵总是不小于自身的熵，这是交叉熵损失函数设计的理论基石。

所以，不等式证明不只是高中数学的练习——它是通往更深层数学推理的必经之路。

### 2. 作差法与作商法——最直接的证明思路

#### 作差法

要证明 $a \geq b$ ，最直接的方式是证明 $a - b \geq 0$ 。这就是 **作差法（Subtraction Method）** ，它的思路极其自然：把两个量"相减"，然后判断差值的符号。

**步骤框架**：

1. 计算 $a - b$
2. 对结果进行化简（通常通过因式分解、配方等手段）
3. 判断化简结果的符号，从而得出 $a$ 与 $b$ 的大小关系

**例题**：已知 $a, b$ 为实数，证明 $a^2 + b^2 \geq 2ab$ 。

我们来作差：

$$
a^2 + b^2 - 2ab = (a - b)^2
$$

> **直觉解读**：任何实数的平方都不小于零，即 $(a - b)^2 \geq 0$ 恒成立。因此 $a^2 + b^2 - 2ab \geq 0$ ，即 $a^2 + b^2 \geq 2ab$ 。等号在 $a = b$ 时成立。

这个看似简单的结论，其实就是后面均值不等式的证明基础。我们用不到三行就完成了严密的推理——这就是作差法的魅力。

#### 作商法

当比较的两个量都是正数时，还可以用 **作商法（Division Method）** ：计算 $\frac{a}{b}$ ，判断它与 $1$ 的大小关系。

**例题**：已知 $a > 0$ ，证明 $a + \frac{1}{a} \geq 2$ 。

我们稍后会用均值不等式来证明这个结论。但如果用作差法：

$$
a + \frac{1}{a} - 2 = \frac{a^2 - 2a + 1}{a} = \frac{(a-1)^2}{a}
$$

> **直觉解读**：分子 $(a-1)^2 \geq 0$ ，分母 $a > 0$ ，因此整个分式 $\geq 0$ 。等号在 $a = 1$ 时成立。

### 3. 均值不等式——不等式世界的基石

现在让我们正式认识高中阶段最重要的不等式之一： **均值不等式（AM-GM Inequality）** ，也叫算术-几何均值不等式。

#### 公式陈述

对于任意两个非负实数 $a \geq 0$ 和 $b \geq 0$ ，有：

$$
\frac{a + b}{2} \geq \sqrt{ab}
$$

> **直觉解读**：这个公式在说——两个非负数的**算术平均值（Arithmetic Mean）** 永远不小于它们的 **几何平均值（Geometric Mean）** 。等号成立的条件是 $a = b$ 。

用日常语言来类比：如果你把一块地分成两部分来种菜，面积分别是 $a$ 和 $b$ ，那么两块地面积的平均值，一定不会比它们面积乘积再开方的结果更小。

#### 证明

证明过程非常优雅，只需要利用我们刚才在作差法中得到的结论。因为 $a, b \geq 0$ ，所以我们可以令 $x = \sqrt{a}$ ， $y = \sqrt{b}$ （ $x, y \geq 0$ ）。则：

$$
x^2 + y^2 \geq 2xy
$$

这正是前面证明过的 $(x - y)^2 \geq 0$ 。将 $x = \sqrt{a}$ ， $y = \sqrt{b}$ 代回：

$$
a + b \geq 2\sqrt{ab}
$$

两边除以 $2$ ：

$$
\frac{a + b}{2} \geq \sqrt{ab}
$$

证毕。等号在 $x = y$ 即 $a = b$ 时成立。

#### 几何解释

均值不等式有一个非常漂亮的几何解释。考虑一个以 $\frac{a+b}{2}$ 为半径的半圆，圆上从直径端点作垂线到半圆上。可以证明：半径（算术平均）永远不短于这条垂线段（几何平均）。

![AM-GM 几何证明](assets/amgm_geometry.png)

> 📌 **图解说明**：线段 $AB$ 的长度为 $a + b$ ，中点 $M$ 到端点的距离为算术平均 $\frac{a+b}{2}$ 。从 $A$ 和 $B$ 的分界点 $P$ 作半圆的垂线到 $H$ ，其长度恰好为几何平均 $\sqrt{ab}$ 。由于半径 $MH$ 总是大于等于垂线 $PH$ ，所以算术平均 $\geq$ 几何平均。你可以运行 `code/plot_amgm_geometry.py` 自行生成这张图。

#### 经典应用：求最值

均值不等式最常见的应用是求函数最值。我们来完成之前提到的例子。

**问题**：当 $a > 0$ 时，求 $f(a) = a + \frac{1}{a}$ 的最小值。

由均值不等式，取 $a$ 和 $\frac{1}{a}$ 作为两个非负数：

$$
a + \frac{1}{a} \geq 2\sqrt{a \cdot \frac{1}{a}} = 2\sqrt{1} = 2
$$

> **直觉解读**：无论 $a$ 取什么正数值， $a + \frac{1}{a}$ 的值都不会小于 $2$ 。当 $a = \frac{1}{a}$ 即 $a = 1$ 时取到等号，最小值为 $2$ 。

⚠️ **使用均值不等式求最值时，需要同时满足三个条件**（口诀：一正二定三相等）：

1. **正**：各项均为正数
2. **定**：和或积中至少一个为定值
3. **相等**：等号条件必须能够取到

### 4. 柯西-施瓦茨不等式——进阶利器

如果说均值不等式是不等式世界的入门基石，那么 **柯西-施瓦茨不等式（Cauchy-Schwarz Inequality）** 就是通往更高层次的桥梁。

#### 基本形式

对于任意实数 $a_1, a_2, b_1, b_2$ ，有：

$$
(a_1^2 + a_2^2)(b_1^2 + b_2^2) \geq (a_1 b_1 + a_2 b_2)^2
$$

> **直觉解读**：两组数各自的"长度的平方"之积，永远不小于它们"对应相乘再求和"的平方。如果你熟悉向量的内积，这其实就是在说 $|\vec{a}|^2 \cdot |\vec{b}|^2 \geq (\vec{a} \cdot \vec{b})^2$ ，即内积不超过两个向量长度之积。

#### 简要证明

利用作差法。展开左右两边之差：

$$
(a_1^2 + a_2^2)(b_1^2 + b_2^2) - (a_1 b_1 + a_2 b_2)^2
$$

$$
= a_1^2 b_2^2 + a_2^2 b_1^2 - 2a_1 b_1 a_2 b_2
$$

$$
= (a_1 b_2 - a_2 b_1)^2 \geq 0
$$

> **直觉解读**：差值恰好是 $(a_1 b_2 - a_2 b_1)^2$ ，即一个完全平方数，必然非负。等号在 $\frac{a_1}{b_1} = \frac{a_2}{b_2}$ （即两组数成比例）时成立。

柯西-施瓦茨不等式在人工智能中意义重大——它直接对应了 **[向量](../../06_向量/)** 内积与余弦相似度的关系。在文本检索、推荐系统中，我们使用余弦相似度来衡量两个向量的相似程度，其理论基础正是这个不等式。

### 5. 不等式证明在人工智能中的影子

你可能会好奇：学这些不等式，将来在人工智能中到底有什么用？让我们提前窥探几个应用场景。

**场景一：损失函数的下界**

在训练机器学习模型时，我们希望最小化损失函数 $L$ 。如果我们能证明 $L \geq 0$ （比如均方误差损失），就知道优化过程有一个理论下界，不会无限下降。这正是不等式证明的思想。

**场景二：正则化与惩罚项**

在 **[正则化（Regularization）](../../../../01_基础能力/02_数学基础/04_最优化/)** 中，我们给目标函数加上约束项 $\lambda \|w\|^2$ 。分析正则化效果时，需要利用不等式来证明加入惩罚项后模型复杂度被有效控制。均值不等式和柯西-施瓦茨不等式在这些推导中频繁出现。

**场景三：概率不等式**

在统计学习理论中，马尔可夫不等式、切比雪夫不等式等概率不等式都需要严格证明。它们用于约束模型预测误差的上界，是理解泛化能力的理论基础。

学好不等式证明的方法论——如何从已知条件出发，通过逻辑推导得到目标结论——将帮助你在未来面对更复杂的数学推理时游刃有余。

## 动手实践

理论讲完了，让我们用 Python 来动手验证刚才学到的不等式。下面的脚本会用数值方法验证 AM-GM 不等式和柯西-施瓦茨不等式，并可视化 $f(a) = a + \frac{1}{a}$ 的函数图像，直观感受最小值的位置。

```python
# 文件：code/inequality_proof_demo.py
# 用途：数值验证 AM-GM 不等式与柯西-施瓦茨不等式，可视化 f(a) = a + 1/a
# 环境要求：Python 3.10+, numpy, matplotlib

import numpy as np

# ========== 1. 验证 AM-GM 不等式 ==========
print("=" * 50)
print("1. AM-GM 不等式验证：(a+b)/2 >= sqrt(a*b)")
print("=" * 50)

rng = np.random.default_rng(42)
test_pairs = rng.uniform(0, 100, size=(5, 2))

for a, b in test_pairs:
    am = (a + b) / 2          # 算术平均
    gm = np.sqrt(a * b)       # 几何平均
    diff = am - gm
    print(f"  a={a:.4f}, b={b:.4f}  =>  AM={am:.4f}, GM={gm:.4f}, AM-GM={diff:.4f} >= 0 ✓")

# 等号条件：a == b
a_eq = 7.0
am_eq = (a_eq + a_eq) / 2
gm_eq = np.sqrt(a_eq * a_eq)
print(f"\n  等号条件 a=b={a_eq}: AM={am_eq:.4f}, GM={gm_eq:.4f}, AM-GM={am_eq - gm_eq:.4f}")

# ========== 2. 验证柯西-施瓦茨不等式 ==========
print("\n" + "=" * 50)
print("2. 柯西-施瓦茨不等式验证：")
print("   (a1²+a2²)(b1²+b2²) >= (a1·b1+a2·b2)²")
print("=" * 50)

for _ in range(5):
    a1, a2 = rng.uniform(-10, 10, size=2)
    b1, b2 = rng.uniform(-10, 10, size=2)
    lhs = (a1**2 + a2**2) * (b1**2 + b2**2)
    rhs = (a1 * b1 + a2 * b2) ** 2
    print(f"  a=({a1:.2f},{a2:.2f}), b=({b1:.2f},{b2:.2f})")
    print(f"    LHS={lhs:.4f}, RHS={rhs:.4f}, LHS-RHS={lhs - rhs:.4f} >= 0 ✓")

# ========== 3. 验证 f(a) = a + 1/a 的最小值 ==========
print("\n" + "=" * 50)
print("3. f(a) = a + 1/a 的最小值（a > 0）")
print("=" * 50)

a_values = np.linspace(0.1, 5, 1000)
f_values = a_values + 1 / a_values
min_idx = np.argmin(f_values)
print(f"  数值最小值: f({a_values[min_idx]:.4f}) = {f_values[min_idx]:.4f}")
print(f"  理论最小值: f(1) = 2")

# ========== 4. 生成函数图像 ==========
try:
    import matplotlib
    matplotlib.use("Agg")
    import matplotlib.pyplot as plt
    import os

    plt.rcParams["font.sans-serif"] = ["DejaVu Sans"]
    plt.rcParams["axes.unicode_minus"] = False

    fig, ax = plt.subplots(figsize=(8, 5))
    ax.plot(a_values, f_values, color="#2196F3", linewidth=2, label=r"$f(a) = a + \frac{1}{a}$")
    ax.axhline(y=2, color="#FF5722", linestyle="--", alpha=0.7, label=r"$y = 2$ (minimum)")
    ax.plot(1, 2, "o", color="#FF5722", markersize=10, zorder=5)
    ax.annotate(
        r"min at $a=1,\ f=2$",
        xy=(1, 2),
        xytext=(2.5, 3.5),
        fontsize=12,
        arrowprops=dict(arrowstyle="->", color="#333"),
        bbox=dict(boxstyle="round,pad=0.3", facecolor="lightyellow", edgecolor="#333"),
    )

    ax.set_xlabel(r"$a$", fontsize=13)
    ax.set_ylabel(r"$f(a)$", fontsize=13)
    ax.set_title(r"$f(a) = a + \frac{1}{a}$ for $a > 0$ (AM-GM lower bound)", fontsize=14)
    ax.set_xlim(0, 5)
    ax.set_ylim(0, 8)
    ax.legend(fontsize=12)
    ax.grid(True, alpha=0.3)
    ax.spines["top"].set_visible(False)
    ax.spines["right"].set_visible(False)

    # 输出到 assets 目录
    script_dir = os.path.dirname(os.path.abspath(__file__))
    assets_dir = os.path.join(script_dir, "..", "assets")

    output_path = os.path.join(assets_dir, "f_a_plus_inv_a.png")
    fig.savefig(output_path, dpi=150, bbox_inches="tight", facecolor="white")
    plt.close(fig)
    print(f"\n  图像已保存到: {output_path}")
except ImportError:
    print("\n  [提示] 未安装 matplotlib，跳过图像生成。")

print("\n所有验证通过！")
```

## 典型误区

| 误区 | 正确理解 |
| --- | --- |
| "验证几个特殊值成立 = 证明" | 代入特殊值只是验证，不是证明。证明必须对**所有**满足条件的值都成立 |
| "均值不等式对任意实数成立" | AM-GM 不等式要求 $a \geq 0$ 且 $b \geq 0$ ，对负数不成立（例如 $a = -1, b = -1$ 时几何平均无意义） |
| "用均值不等式求最值时忘记检查等号条件" | 必须验证等号 $a = b$ 的条件**能够取到**。如果在约束条件下 $a = b$ 不可能发生，则不能用 AM-GM 得出最值 |
| "作差法只能处理简单表达式" | 作差法非常通用，配合因式分解、配方法可以处理复杂的不等式证明 |
| "柯西-施瓦茨不等式只在高等数学中有用" | 它在高中数学竞赛、向量内积分析、人工智能的余弦相似度计算中都有直接应用 |

## 练习题

### 练习 1：作差法基础（难度：⭐）

证明：对于任意实数 $x$ ，有 $x^2 + 1 \geq 2x$ 。

<details>
<summary>💡 提示</summary>

尝试计算 $x^2 + 1 - 2x$ ，看看能不能把它写成某个完全平方的形式。

</details>

<details>
<summary>✅ 参考答案</summary>

$$x^2 + 1 - 2x = (x - 1)^2 \geq 0$$

∴ $x^2 + 1 \geq 2x$ 。等号在 $x = 1$ 时成立。

</details>

### 练习 2：均值不等式应用（难度：⭐⭐）

已知 $x > 0$ ，求 $f(x) = 4x + \frac{9}{x}$ 的最小值，并指出取到最小值时 $x$ 的值。

<details>
<summary>💡 提示</summary>

将 $4x$ 和 $\dfrac{9}{x}$ 视为均值不等式中的两个正数，利用 $a + b \geq 2\sqrt{ab}$ 。注意检查等号条件 $4x = \dfrac{9}{x}$ 是否有正数解。

</details>

<details>
<summary>✅ 参考答案</summary>

由均值不等式：

$$4x + \dfrac{9}{x} \geq 2\sqrt{4x \cdot \dfrac{9}{x}} = 2\sqrt{36} = 12$$

等号条件为 $4x = \dfrac{9}{x}$ ，即 $4x^2 = 9$ ，解得 $x = \dfrac{3}{2}$ （取正值）。

∴ $f(x)$ 的最小值为 $12$ ，在 $x = \dfrac{3}{2}$ 时取到。

</details>

### 练习 3：柯西-施瓦茨不等式应用（难度：⭐⭐⭐）

已知 $a^2 + b^2 = 1$ ，证明 $|3a + 4b| \leq 5$ 。

<details>
<summary>💡 提示</summary>

尝试用柯西-施瓦茨不等式 $(a_1^2 + a_2^2)(b_1^2 + b_2^2) \geq (a_1 b_1 + a_2 b_2)^2$ ，令 $a_1 = a$ ， $a_2 = b$ ， $b_1 = 3$ ， $b_2 = 4$ 。

</details>

<details>
<summary>✅ 参考答案</summary>

由柯西-施瓦茨不等式：

$$(a^2 + b^2)(3^2 + 4^2) \geq (3a + 4b)^2$$

∵ $a^2 + b^2 = 1$ ，∴：

$$1 \times 25 \geq (3a + 4b)^2$$

即 $(3a + 4b)^2 \leq 25$ ，∴ $|3a + 4b| \leq 5$ 。

等号在 $\dfrac{a}{3} = \dfrac{b}{4}$ 即 $a = \dfrac{3}{5}$ ， $b = \dfrac{4}{5}$ 时成立。

</details>

### 练习 4：综合证明（难度：⭐⭐）

已知 $a, b > 0$ 且 $a + b = 1$ ，证明 $\frac{1}{a} + \frac{1}{b} \geq 4$ 。

<details>
<summary>💡 提示</summary>

方法一：利用均值不等式先求 $ab$ 的最大值，再求 $\dfrac{1}{a} + \dfrac{1}{b} = \dfrac{a+b}{ab}$ 的最小值。

方法二：直接利用柯西-施瓦茨不等式 $(a + b)\left(\dfrac{1}{a} + \dfrac{1}{b}\right) \geq (1 + 1)^2 = 4$ 。

</details>

<details>
<summary>✅ 参考答案</summary>

**方法一**（均值不等式）：

$$\dfrac{1}{a} + \dfrac{1}{b} = \dfrac{a + b}{ab} = \dfrac{1}{ab}$$

由 AM-GM 不等式： $\dfrac{a+b}{2} \geq \sqrt{ab}$ ，即 $\dfrac{1}{2} \geq \sqrt{ab}$ ，所以 $ab \leq \dfrac{1}{4}$ 。 ∴ $\dfrac{1}{ab} \geq 4$ 。等号在 $a = b = \dfrac{1}{2}$ 时成立。

**方法二**（柯西-施瓦茨不等式）：

$$(a + b)\left(\dfrac{1}{a} + \dfrac{1}{b}\right) \geq \left(\sqrt{a \cdot \dfrac{1}{a}} + \sqrt{b \cdot \dfrac{1}{b}}\right)^2 = (1 + 1)^2 = 4$$

∵ $a + b = 1$ ，∴ $\dfrac{1}{a} + \dfrac{1}{b} \geq 4$ 。

</details>
