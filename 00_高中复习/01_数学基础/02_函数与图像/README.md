# 函数与图像

> **所属路径**：`00_高中复习/01_数学基础/02_函数与图像`
> **包含知识点**：5 个

---

## 概述

函数是数学中最核心的概念之一，它描述了变量之间的对应关系。在人工智能领域，神经网络的每一层本质上就是一个函数——接收输入，经过运算，产生输出。理解函数的基本性质（定义域、值域、单调性、奇偶性）和图像变换规律，是后续学习指数函数、对数函数、三角函数以及微积分的必要基础。

本主题从函数的基本概念出发，依次探讨函数的"合法输入与可能输出"（定义域与值域）、"上升与下降、对称与非对称"（单调性与奇偶性）、"重复与镜像"（周期性与对称性）、"如何从简单函数快速推导出复杂函数的图像"（图像平移与变换），最后通过反函数与复合函数理解"函数的逆操作与嵌套组合"。五个知识点层层递进，帮助你建立起函数分析的完整工具箱。

## 学习建议

1. **学习顺序**：建议按 01 → 02 → 03 → 04 → 05 的顺序学习。定义域与值域是理解函数的基础，单调性与奇偶性需要在此基础上分析函数的趋势和对称性，周期性与对称性进一步探讨函数的重复规律和对称结构，图像变换综合运用前面所学知识来快速分析复杂函数，反函数与复合函数则帮助你理解函数的"逆操作"与"嵌套组合"。
2. **时间分配**：每个知识点预计 50–55 分钟，整个主题约 3.5–4 小时。
3. **常见困难**：水平方向的变换（左右平移、水平伸缩）往往违反直觉——"减法对应右移""乘以大于 1 的数对应压缩"。建议遇到困惑时多画图验证，从具体数值的代入中建立直觉。
4. **动手实践**：每节课都附带了 Python 代码示例，强烈建议亲手运行，通过可视化加深理解。
