## 正文讲解

### 1. 向量方法的威力

传统几何证明常常需要巧妙的辅助线和独特的思路，不同的题目可能需要完全不同的证法。而向量方法提供了一种 **统一的、系统化的** 解题策略——只需要把几何条件翻译成向量语言，然后进行代数运算，最后把代数结果翻译回几何结论。

这种"几何 → 代数 → 计算 → 几何"的思路，其实和人工智能处理问题的方式非常相似：AI 把图像、文本等非结构化数据"翻译"成数字向量，在向量空间中进行计算，最后将结果"翻译"回人类能理解的输出。

### 2. 证明平行——共线条件的应用

两条线段平行，在向量语言中就是它们对应的方向向量共线。回顾一下之前学过的判定方法：

**两个非零向量 $\vec{a}$ 和 $\vec{b}$ 平行的充要条件**：

- **方法一（数乘法）**：存在实数 $\lambda$ 使得 $\vec{b} = \lambda \vec{a}$
- **方法二（坐标交叉法）**：若 $\vec{a} = (a_1, a_2)$ ， $\vec{b} = (b_1, b_2)$ ，则 $a_1 b_2 - a_2 b_1 = 0$

来看一个经典例题：**证明三角形中位线平行于第三边且等于第三边的一半。**

设三角形 $ABC$ 中， $M$ 和 $N$ 分别是 $AB$ 和 $AC$ 的中点。用向量来证明 $MN \parallel BC$ 且 $|MN| = \dfrac{1}{2}|BC|$ 。

设 $\overrightarrow{AB} = \vec{b}$ ， $\overrightarrow{AC} = \vec{c}$ ，则：

$$
\overrightarrow{AM} = \frac{1}{2}\vec{b}, \quad \overrightarrow{AN} = \frac{1}{2}\vec{c}
$$

$$
\overrightarrow{MN} = \overrightarrow{AN} - \overrightarrow{AM} = \frac{1}{2}\vec{c} - \frac{1}{2}\vec{b} = \frac{1}{2}(\vec{c} - \vec{b})
$$

又 $\overrightarrow{BC} = \vec{c} - \vec{b}$ ，所以：

$$
\overrightarrow{MN} = \frac{1}{2}\overrightarrow{BC}
$$

∵ $\overrightarrow{MN} = \dfrac{1}{2}\overrightarrow{BC}$

∴ $MN \parallel BC$ 且 $|MN| = \dfrac{1}{2}|BC|$ 。 $\square$

注意这个证明多么简洁——没有辅助线，没有巧妙构思，只是老老实实地用向量运算。

### 3. 证明垂直——数量积为零

两条线段垂直，在向量语言中就是它们对应的方向向量的 **[数量积（Dot Product）](../02_数量积/02_数量积.md)** 为零。

**例题**：证明正方形的两条对角线互相垂直。

设正方形 $ABCD$ 中 $\overrightarrow{AB} = \vec{a}$ ， $\overrightarrow{AD} = \vec{b}$ 。由于是正方形，有 $|\vec{a}| = |\vec{b}|$ 且 $\vec{a} \cdot \vec{b} = 0$ 。

两条对角线为：

$$
\overrightarrow{AC} = \vec{a} + \vec{b}, \quad \overrightarrow{BD} = \vec{b} - \vec{a}
$$

计算它们的数量积：

$$
\overrightarrow{AC} \cdot \overrightarrow{BD} = (\vec{a} + \vec{b}) \cdot (\vec{b} - \vec{a}) = \vec{a} \cdot \vec{b} - \vec{a} \cdot \vec{a} + \vec{b} \cdot \vec{b} - \vec{b} \cdot \vec{a}
$$

$$
= \vec{b} \cdot \vec{b} - \vec{a} \cdot \vec{a} = |\vec{b}|^2 - |\vec{a}|^2 = 0
$$

∵ $\overrightarrow{AC} \cdot \overrightarrow{BD} = 0$

∴ $AC \perp BD$ 。 $\square$

这个证明的关键在于利用了 $|\vec{a}| = |\vec{b}|$ 这一正方形的性质。事实上，更一般地说，这证明了**菱形的对角线互相垂直**——因为证明中只用到了"邻边等长"这一条件。

### 4. 面积计算——向量的"交叉相乘"公式

> ⚠️ **超纲提示**：这里用到的 $a_1 b_2 - a_2 b_1$ 在大学线性代数中叫做"2×2 行列式"或"叉积的二维形式"，中国高中课程标准不作要求。但这个公式本质上只是"交叉相乘再相减"，理解起来并不困难，而且它是计算机图形学和人工智能中计算面积的基本工具。

在前面的课程中我们学过，两个二维向量 $\vec{a} = (a_1, a_2)$ 和 $\vec{b} = (b_1, b_2)$ 可以通过以下公式计算面积。定义 $D = a_1 b_2 - a_2 b_1$ ，其绝对值就是平行四边形的面积。

因此：

**平行四边形面积**：

$$
S_{\text{平行四边形}} = |a_1 b_2 - a_2 b_1|
$$

**三角形面积**（平行四边形面积的一半）：

$$
S_{\triangle} = \frac{1}{2} |a_1 b_2 - a_2 b_1|
$$

如果三角形的三个顶点是 $A(x_1, y_1)$ 、 $B(x_2, y_2)$ 、 $C(x_3, y_3)$ ，那么：

$$
S_{\triangle ABC} = \frac{1}{2} |(x_2 - x_1)(y_3 - y_1) - (x_3 - x_1)(y_2 - y_1)|
$$

> **直觉解读**：这个公式把 $\overrightarrow{AB}$ 和 $\overrightarrow{AC}$ 的坐标代入"交叉相乘"公式。它在计算机图形学中被广泛使用——判断一个点在三角形内部还是外部，就是通过比较面积来实现的。

### 5. 直线的参数方程——向量的动态描述

向量不仅能描述静态的几何关系，还能用来描述 **运动的轨迹** 。

设直线经过点 $P_0(x_0, y_0)$ ，方向向量为 $\vec{d} = (d_1, d_2)$ 。直线上任意一点 $P$ 的位置向量可以写成：

$$
\overrightarrow{OP} = \overrightarrow{OP_0} + t \vec{d}
$$

其中 $t$ 是一个实数参数。当 $t$ 取遍所有实数时， $P$ 跑遍整条直线。用坐标表示就是：

$$
\begin{cases} x = x_0 + t d_1 \\ y = y_0 + t d_2 \end{cases} \quad (t \in \mathbb{R})
$$

这就是直线的 **参数方程（Parametric Equation）** 。

几个特殊情况：
- $t = 0$ 时， $P = P_0$ （起点）
- $t = 1$ 时， $P = P_0 + \vec{d}$ （沿方向移动一个单位）
- $0 \leq t \leq 1$ 时， $P$ 在线段 $P_0$ 到 $P_0 + \vec{d}$ 上

参数方程在计算机图形学、机器人路径规划和光线追踪（Ray Tracing）中非常常用。一条光线就是一个"起点 + 方向"的参数方程，AI 渲染引擎用它来模拟光的传播。

### 6. 解题策略总结

使用向量方法解决几何问题，一般遵循以下步骤：

```mermaid
flowchart LR
    A[建立向量] --> B[翻译条件]
    B --> C[代数运算]
    C --> D[几何结论]
```

> 📌 **图解说明**：向量法解题的四步流程——先选取基底、表示关键向量，再将几何条件转化为向量等式，进行代数化简后得到几何结论。

具体来说：

1. **建立向量**：选择合适的基底向量（通常选取从同一顶点出发的两条边），用基底表示所有相关向量
2. **翻译条件**：平行 → 共线（数乘关系）；垂直 → 数量积为零；中点 → 系数为 $\dfrac{1}{2}$
3. **代数运算**：展开、化简、利用已知条件
4. **几何结论**：把代数结果翻译回几何语言

## 动手实践

让我们用 Python 来验证三角形中位线定理和面积公式：

```python
# 文件：code/vector_geometry.py
# 用向量方法验证几何定理并计算面积
# 环境要求：Python 3.10+, numpy

import numpy as np

# ---- 验证三角形中位线定理 ----
A = np.array([0, 0])
B = np.array([6, 0])
C = np.array([2, 4])

# M 和 N 分别是 AB 和 AC 的中点
M = (A + B) / 2
N = (A + C) / 2

MN = N - M
BC = C - B

print("=== 三角形中位线定理 ===")
print(f"A = {A}, B = {B}, C = {C}")
print(f"M (AB中点) = {M}, N (AC中点) = {N}")
print(f"MN = {MN}")
print(f"BC = {BC}")
print(f"MN = 0.5 * BC? {np.allclose(MN, 0.5 * BC)}")

# ---- 三角形面积计算 ----
AB = B - A
AC = C - A
area = 0.5 * abs(AB[0] * AC[1] - AB[1] * AC[0])
print(f"\n=== 三角形面积 ===")
print(f"AB = {AB}, AC = {AC}")
print(f"面积 = {area}")

# ---- 直线参数方程 ----
P0 = np.array([1, 2])  # 起点
d = np.array([3, 1])   # 方向向量
print(f"\n=== 直线参数方程 ===")
print(f"起点 P0 = {P0}, 方向 d = {d}")
print("直线上的点（t = -1, 0, 1, 2）:")
for t in [-1, 0, 1, 2]:
    P = P0 + t * d
    print(f"  t = {t}: P = {P}")

# ---- 垂直判定 ----
v1 = np.array([3, 4])
v2 = np.array([4, -3])
dot = np.dot(v1, v2)
print(f"\n=== 垂直判定 ===")
print(f"v1 = {v1}, v2 = {v2}")
print(f"v1 · v2 = {dot}")
print(f"v1 ⊥ v2? {dot == 0}")
```

**运行说明**：
- 环境要求：Python 3.10+, numpy
- 运行命令：`python code/vector_geometry.py`

**预期输出**：
```
=== 三角形中位线定理 ===
A = [0 0], B = [6 0], C = [2 4]
M (AB中点) = [3. 0.], N (AC中点) = [1. 2.]
MN = [-2.  2.]
BC = [-4  4]
MN = 0.5 * BC? True

=== 三角形面积 ===
AB = [6 0], AC = [2 4]
面积 = 12.0

=== 直线参数方程 ===
起点 P0 = [1 2], 方向 d = [3 1]
直线上的点（t = -1, 0, 1, 2）:
  t = -1: P = [-2  1]
  t = 0: P = [1 2]
  t = 1: P = [4 3]
  t = 2: P = [7 4]

=== 垂直判定 ===
v1 = [3 4], v2 = [ 4 -3]
v1 · v2 = 0
v1 ⊥ v2? True
```

代码验证了中位线定理（ $\overrightarrow{MN} = 0.5 \times \overrightarrow{BC}$ ），演示了面积计算、参数方程求点，以及垂直判定。

## 典型误区

| 误区 | 正确理解 |
| --- | --- |
| 证明平行只需要说明方向相同 | 向量平行包括同向和反向两种情况。 $\vec{b} = \lambda \vec{a}$ 中 $\lambda$ 可正可负 |
| 面积公式 $a_1 b_2 - a_2 b_1$ 结果为负说明计算错了 | 交叉相乘的结果可以为负，面积取绝对值即可。负号只表示向量的相对朝向 |
| 参数方程中 $t$ 只能取正值 | $t$ 可以是任意实数。负值表示沿方向向量的反方向移动 |
| 用向量证题时可以随意选择基底 | 基底选择会影响计算复杂度。通常选择题目中自然出现的向量（如三角形的两条边）作为基底 |

## 练习题

### 练习 1：面积计算（难度：⭐）

计算以 $A(1, 2)$ ， $B(4, 1)$ ， $C(2, 5)$ 为顶点的三角形的面积。

<details>
<summary>💡 提示</summary>

计算 $\overrightarrow{AB}$ 和 $\overrightarrow{AC}$ ，然后用公式 $S = \dfrac{1}{2}|x_1 y_2 - x_2 y_1|$ 。

</details>

<details>
<summary>✅ 参考答案</summary>

$\overrightarrow{AB} = (3, -1)$ ， $\overrightarrow{AC} = (1, 3)$

$$S = \dfrac{1}{2}|3 \times 3 - (-1) \times 1| = \dfrac{1}{2}|9 + 1| = 5$$

</details>

### 练习 2：垂直证明（难度：⭐⭐）

在等腰三角形 $ABC$ 中， $AB = AC$ ， $M$ 是 $BC$ 的中点。用向量方法证明 $AM \perp BC$ 。

<details>
<summary>💡 提示</summary>

设 $\overrightarrow{AB} = \vec{b}$ ， $\overrightarrow{AC} = \vec{c}$ ，用向量表示 $\overrightarrow{AM}$ 和 $\overrightarrow{BC}$ ，计算数量积。利用 $|\vec{b}| = |\vec{c}|$ 。

</details>

<details>
<summary>✅ 参考答案</summary>

设 $\overrightarrow{AB} = \vec{b}$ ， $\overrightarrow{AC} = \vec{c}$ 。

$M$ 是 $BC$ 中点，所以 $\overrightarrow{AM} = \dfrac{1}{2}(\vec{b} + \vec{c})$

$\overrightarrow{BC} = \vec{c} - \vec{b}$

$$\overrightarrow{AM} \cdot \overrightarrow{BC} = \dfrac{1}{2}(\vec{b} + \vec{c}) \cdot (\vec{c} - \vec{b}) = \dfrac{1}{2}(|\vec{c}|^2 - |\vec{b}|^2)$$

∵ $AB = AC$ ，即 $|\vec{b}| = |\vec{c}|$

∴ $\overrightarrow{AM} \cdot \overrightarrow{BC} = \dfrac{1}{2}(|\vec{c}|^2 - |\vec{b}|^2) = 0$

∴ $AM \perp BC$ 。 $\square$

</details>

### 练习 3：参数方程（难度：⭐⭐）

直线经过点 $P(2, 3)$ ，方向向量为 $\vec{d} = (1, -2)$ 。
(a) 写出直线的参数方程。
(b) 求直线上 $t = 3$ 对应的点。
(c) 判断点 $Q(5, -3)$ 是否在该直线上。

<details>
<summary>💡 提示</summary>

参数方程为 $(x, y) = (2, 3) + t(1, -2)$ 。对于 (c)，检查是否存在 $t$ 使得两个分量都满足。

</details>

<details>
<summary>✅ 参考答案</summary>

(a) 参数方程：

$\begin{cases} x = 2 + t \\ y = 3 - 2t \end{cases} \quad (t \in \mathbb{R})$

(b) 当 $t = 3$ 时：

$x = 2 + 3 = 5$ ， $y = 3 - 6 = -3$

对应点为 $(5, -3)$

(c) 检验 $Q(5, -3)$ ：

由 $x = 2 + t = 5$ 得 $t = 3$

由 $y = 3 - 2t = 3 - 6 = -3$ ✓

∵ 两个分量在 $t = 3$ 时同时满足

∴ $Q(5, -3)$ 在直线上。

</details>
