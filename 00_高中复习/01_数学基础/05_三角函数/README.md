# 三角函数

> **所属路径**：`00_高中复习/01_数学基础/05_三角函数`
> **包含知识点**：5 个

---

## 概述

三角函数是连接几何与代数的桥梁，也是信号处理、物理模拟和人工智能的基础工具。它以"角度与比值"的关系为核心，具有独特的周期性和对称性。在 AI 领域，三角函数的应用远超高中数学的范畴：Transformer 的位置编码使用正弦和余弦函数为序列赋予位置信息，旋转位置编码（RoPE）利用和差角公式编码相对距离，余弦相似度是衡量向量相似性的标准度量。

本主题从弧度制和三角比的基本定义出发，通过三角函数图像建立视觉直觉，掌握恒等变换的化简技巧，最后通过正弦定理和余弦定理将三角函数应用于一般三角形的计算。

## 学习建议

1. **学习顺序**：建议按 01 → 02 → 03 → 04 → 05 的顺序学习。弧度与三角比是一切的基础，图像帮助建立直觉，恒等变换提供计算工具，正弦/余弦定理处理一般三角形，解三角形是综合应用。
2. **时间分配**：每个知识点预计 45-55 分钟，整个主题约 4 小时。
3. **常见困难**：恒等变换的公式较多，容易混淆。建议理解推导过程（所有公式都可以从和差角公式推出），而非死记硬背。解三角形中的 SSA 歧义情况是难点，务必通过具体例子加深理解。
