## 正文讲解

### 1. 为什么需要恒等变换

在 **[弧度与三角比](../01_弧度与三角比/01_弧度与三角比.md)** 中，我们学到了最基本的恒等式 $\sin^2\theta + \cos^2\theta = 1$ 。但实际问题中，我们经常遇到更复杂的三角表达式——比如 $\sin(A + B)$ 、 $\cos 2\theta$ 、 $\sin\alpha + \cos\alpha$ 等。恒等变换就是一套"化简工具箱"，帮助我们把复杂的表达式变成更容易处理的形式。

在 AI 中，恒等变换的思想同样重要。比如 **旋转位置编码（RoPE）** 利用和差角公式来将位置信息编码到注意力计算中，使模型能感知 token 之间的相对距离。

### 2. 和差角公式

**和差角公式（Sum and Difference Formulas）** 解决的问题是：已知 $\sin A$ 、 $\cos A$ 、 $\sin B$ 、 $\cos B$ ，求 $\sin(A \pm B)$ 和 $\cos(A \pm B)$ 。

$$
\sin(A + B) = \sin A \cos B + \cos A \sin B
$$

$$
\sin(A - B) = \sin A \cos B - \cos A \sin B
$$

$$
\cos(A + B) = \cos A \cos B - \sin A \sin B
$$

$$
\cos(A - B) = \cos A \cos B + \sin A \sin B
$$

$$
\tan(A + B) = \frac{\tan A + \tan B}{1 - \tan A \tan B}
$$

这些公式看起来很多，但它们有一个优美的几何来源。下面这张图展示了如何通过单位圆上的几何关系推导和角公式：

![和角公式的几何证明](assets/sum_angle_proof.png)

> 📌 **图解说明**：在单位圆上，角 $\alpha$ 对应点 $P$ ，角 $\alpha + \beta$ 对应点 $Q$ 。通过将 $Q$ 向 $OP$ 方向投影到 $H$ ，再分别向坐标轴投影，就能用 $\alpha$ 和 $\beta$ 的三角函数表示 $Q$ 的坐标——这正是和角公式。你可以运行 `code/plot_sum_angle.py` 自行生成这张图。

> **记忆技巧**：
> - 正弦的和差公式是"正余余正"——$\sin$ 配 $\cos$ 交叉相乘
> - 余弦的和差公式是"余余正正"——同类函数相乘
> - 正弦公式的符号与中间"±"**相同**，余弦公式的符号与中间"±"**相反**

让我们验证一下。已知 $\sin 30° = \dfrac{1}{2}$ ， $\cos 30° = \dfrac{\sqrt{3}}{2}$ ， $\sin 45° = \cos 45° = \dfrac{\sqrt{2}}{2}$ ，求 $\sin 75°$ ：

$$
\sin 75° = \sin(30° + 45°) = \frac{1}{2} \cdot \frac{\sqrt{2}}{2} + \frac{\sqrt{3}}{2} \cdot \frac{\sqrt{2}}{2} = \frac{\sqrt{2} + \sqrt{6}}{4}
$$

### 3. 倍角公式

当和差角公式中 $A = B$ 时，就得到了 **倍角公式（Double Angle Formulas）**：

$$
\sin 2\theta = 2\sin\theta \cos\theta
$$

$$
\cos 2\theta = \cos^2\theta - \sin^2\theta = 2\cos^2\theta - 1 = 1 - 2\sin^2\theta
$$

$$
\tan 2\theta = \frac{2\tan\theta}{1 - \tan^2\theta}
$$

> **直觉解读**：倍角公式把"双倍角的三角函数"用"原角的三角函数"来表示。这在积分、化简和降幂时非常有用。 $\cos 2\theta$ 有三种等价形式，根据需要选择最方便的那个。

### 4. 半角公式

从 $\cos 2\theta$ 的倍角公式反推，可以得到 **半角公式（Half Angle Formulas）**。把 $\cos 2\theta = 2\cos^2\theta - 1$ 改写：

$$
\cos^2\theta = \frac{1 + \cos 2\theta}{2}
$$

$$
\sin^2\theta = \frac{1 - \cos 2\theta}{2}
$$

> **直觉解读**：半角公式也叫"**降幂公式**"——它把 $\sin^2$ 或 $\cos^2$ 这样的"平方"（二次）表达式，降为一次的 $\cos$ 表达式。在微积分中对 $\sin^2 x$ 积分时，这个公式是关键工具。

### 5. 和差化积与积化和差

**和差化积**（Sum to Product）：

$$
\sin A + \sin B = 2\sin\frac{A+B}{2}\cos\frac{A-B}{2}
$$

$$
\cos A + \cos B = 2\cos\frac{A+B}{2}\cos\frac{A-B}{2}
$$

**积化和差**（Product to Sum）：

$$
\sin A \cos B = \frac{1}{2}[\sin(A+B) + \sin(A-B)]
$$

$$
\cos A \cos B = \frac{1}{2}[\cos(A-B) + \cos(A+B)]
$$

> 📌 这些公式在信号处理中用途广泛——"频率混合"本质上就是三角函数的乘积，而积化和差将其分解为不同频率分量的叠加。

### 6. 辅助角公式

在实际问题中，经常遇到 $a\sin x + b\cos x$ 这种"混合"表达式。辅助角公式可以把它统一为一个正弦函数：

$$
a\sin x + b\cos x = \sqrt{a^2 + b^2} \sin(x + \varphi)
$$

其中 $\tan\varphi = \dfrac{b}{a}$ 。

> **直觉解读**： $a\sin x + b\cos x$ 看起来是两个波的叠加，但实际上它**仍然是一个正弦波**——只是振幅和相位发生了变化。辅助角公式揭示了这一点：振幅变为 $\sqrt{a^2 + b^2}$ ，相位偏移了 $\varphi$ 。

**例题**：把 $\sin x + \sqrt{3}\cos x$ 化为 $R\sin(x + \varphi)$ 的形式。

解： $a = 1$ ， $b = \sqrt{3}$ ，所以：

$$
R = \sqrt{1^2 + (\sqrt{3})^2} = \sqrt{4} = 2
$$

$$
\tan\varphi = \frac{\sqrt{3}}{1} = \sqrt{3} \implies \varphi = \frac{\pi}{3}
$$

$$
\sin x + \sqrt{3}\cos x = 2\sin\left(x + \frac{\pi}{3}\right)
$$

## 动手实践

```python
# 文件：code/trig_identities.py
# 三角恒等变换的数值验证
# 环境要求：Python 3.10+（仅使用标准库 math）

import math


def verify(name: str, left: float, right: float, tol: float = 1e-10):
    """验证恒等式左右两边是否相等"""
    ok = abs(left - right) < tol
    status = "✓" if ok else "✗"
    print(f"  {status} {name}: 左 = {left:.8f}, 右 = {right:.8f}")
    return ok


if __name__ == "__main__":
    A = math.radians(37)  # 选一个非特殊角做验证
    B = math.radians(53)

    print("=" * 55)
    print("和差角公式验证（A=37°, B=53°）")
    print("=" * 55)
    verify("sin(A+B) = sinA cosB + cosA sinB",
           math.sin(A + B),
           math.sin(A)*math.cos(B) + math.cos(A)*math.sin(B))
    verify("cos(A+B) = cosA cosB - sinA sinB",
           math.cos(A + B),
           math.cos(A)*math.cos(B) - math.sin(A)*math.sin(B))
    verify("sin(A-B) = sinA cosB - cosA sinB",
           math.sin(A - B),
           math.sin(A)*math.cos(B) - math.cos(A)*math.sin(B))

    print("\n" + "=" * 55)
    print("倍角公式验证（θ=37°）")
    print("=" * 55)
    theta = A
    verify("sin(2θ) = 2sinθ cosθ",
           math.sin(2*theta),
           2*math.sin(theta)*math.cos(theta))
    verify("cos(2θ) = cos²θ - sin²θ",
           math.cos(2*theta),
           math.cos(theta)**2 - math.sin(theta)**2)
    verify("cos(2θ) = 2cos²θ - 1",
           math.cos(2*theta),
           2*math.cos(theta)**2 - 1)

    print("\n" + "=" * 55)
    print("半角（降幂）公式验证")
    print("=" * 55)
    verify("sin²θ = (1-cos2θ)/2",
           math.sin(theta)**2,
           (1 - math.cos(2*theta)) / 2)
    verify("cos²θ = (1+cos2θ)/2",
           math.cos(theta)**2,
           (1 + math.cos(2*theta)) / 2)

    print("\n" + "=" * 55)
    print("辅助角公式验证：sinx + √3·cosx = 2sin(x+π/3)")
    print("=" * 55)
    for deg in [0, 30, 45, 60, 90, 120, 180]:
        x = math.radians(deg)
        left = math.sin(x) + math.sqrt(3) * math.cos(x)
        right = 2 * math.sin(x + math.pi/3)
        verify(f"x = {deg}°", left, right)
```

**运行说明**：
- 环境要求：Python 3.10+（仅使用标准库 `math`）
- 运行命令：`python code/trig_identities.py`

## 典型误区

| 误区 | 正确理解 |
| ---- | -------- |
| $\cos(A+B)$ 的符号记反 | 余弦的和差公式中间的符号和 $\pm$ **相反**： $\cos(A+B)$ 中间是**减号** |
| 倍角公式 $\cos 2\theta$ 只记一种形式 | $\cos 2\theta$ 有三种等价形式，根据已知条件选择最方便的 |
| 辅助角公式中 $\varphi$ 的象限搞错 | $\tan\varphi = b/a$ 有两个解（相差 $\pi$ ），需要根据 $a$ 和 $b$ 的正负确定 $\varphi$ 所在象限 |
| 认为恒等式需要死记硬背 | 所有公式都可以从和差角公式推导出来。理解推导过程比硬记更可靠 |

## 练习题

### 练习 1：和差角公式应用（难度：⭐）

利用和差角公式求 $\cos 15°$ 的精确值。

<details>
<summary>💡 提示</summary>

$15° = 45° - 30°$ ，使用 $\cos(A-B) = \cos A \cos B + \sin A \sin B$ 。

</details>

<details>
<summary>✅ 参考答案</summary>

$$\cos 15° = \cos(45° - 30°) = \cos 45° \cos 30° + \sin 45° \sin 30°$$$$= \dfrac{\sqrt{2}}{2} \cdot \dfrac{\sqrt{3}}{2} + \dfrac{\sqrt{2}}{2} \cdot \dfrac{1}{2} = \dfrac{\sqrt{6} + \sqrt{2}}{4}$$

</details>

### 练习 2：倍角与降幂（难度：⭐⭐）

化简 $\cos^4 x - \sin^4 x$ 。

<details>
<summary>💡 提示</summary>

利用平方差分解 $a^4 - b^4 = (a^2 + b^2)(a^2 - b^2)$ ，然后利用勾股恒等式和倍角公式。

</details>

<details>
<summary>✅ 参考答案</summary>

$$\cos^4 x - \sin^4 x = (\cos^2 x + \sin^2 x)(\cos^2 x - \sin^2 x) = 1 \cdot \cos 2x = \cos 2x$$

</details>

### 练习 3：辅助角公式（难度：⭐⭐）

将 $3\sin x + 4\cos x$ 化为 $R\sin(x + \varphi)$ 的形式，并求其最大值。

<details>
<summary>💡 提示</summary>

$R = \sqrt{a^2 + b^2}$ ， $\tan\varphi = \dfrac{b}{a}$ 。

</details>

<details>
<summary>✅ 参考答案</summary>

$R = \sqrt{3^2 + 4^2} = \sqrt{25} = 5$

$\tan\varphi = \dfrac{4}{3}$ ， $\varphi = \arctan\dfrac{4}{3}$

$$3\sin x + 4\cos x = 5\sin\left(x + \arctan\dfrac{4}{3}\right)$$

最大值为 $R = 5$ ，当 $x + \varphi = \dfrac{\pi}{2}$ 时取到。

</details>
