# 排列组合

> **所属路径**：`00_高中复习/01_数学基础/08_排列组合`
> **包含知识点**：4 个

---

## 概述

排列组合是研究"如何数清楚"的数学分支。在日常生活中，我们会遇到各种选择和安排的问题：密码有多少种设法？从一群人中选委员会有多少种方案？这些看似简单的问题，背后都需要系统的计数方法。

在人工智能领域，排列组合的重要性远超高中考试的范围。机器学习中的特征选择需要从 $n$ 个特征中挑选 $k$ 个最佳子集——这正是组合数 $C(n,k)$ 描述的问题；超参数搜索需要枚举所有可能的参数组合；模型集成（Ensemble）需要计算子模型的组合方式；而二项式定理则是理解概率分布（如二项分布）的数学基础。掌握排列组合，你就拥有了"有条理地穷举所有可能性"的思维工具。

本主题从最基础的加法原理和乘法原理出发，建立分类与分步的计数直觉；接着引入排列和组合的公式化工具；然后通过二项式定理将组合数与代数展开联系起来；最后学习常见的计数模型，处理更复杂的实际问题。

## 学习建议

1. **学习顺序**：建议按 01 → 02 → 03 → 04 的顺序学习。加法乘法原理是所有计数问题的思维基础，排列组合公式将基本原理工具化，二项式定理是组合数的重要应用，常见计数模型则是对前三者的综合运用。
2. **时间分配**：每个知识点预计 40–50 分钟，整个主题约 3 小时。
3. **常见困难**：排列和组合容易混淆（关键区分："顺序是否重要"），建议每次做题前先明确这一点。计数问题容易重复计数或遗漏，画树形图和列表是最好的检验方法。
