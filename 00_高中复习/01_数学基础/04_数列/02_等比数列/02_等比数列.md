## 正文讲解

### 1. 从"翻倍增长"说起

在 **[等差数列](../01_等差数列/01_等差数列.md)** 中，我们研究的是"每步加一个固定值"的增长模式。但现实中还有一种更"猛烈"的增长方式——**每步乘一个固定倍数**。

想象一个经典场景：你把 1000 元存入银行，年利率 $10\%$ 。每年底的存款金额是：

$$
1000, \; 1100, \; 1210, \; 1331, \; 1464.1, \; \ldots
$$

每一项都是前一项的 $1.1$ 倍——这就是一个 **等比数列（Geometric Sequence）**。"等比"的意思是"比值相等"，每一项除以前一项都得到同一个数。

在人工智能中，等比数列的身影更是无处不在：

- **学习率指数衰减**：每个 epoch 学习率乘以 $0.95$ ，形成等比数列
- **权重初始化**：如果权重每层放大 $1.1$ 倍，经过 100 层后会变成 $1.1^{100} \approx 13781$ 倍——这就是 **[梯度爆炸](../../../02_核心原理/03_深度学习/02_反向传播/03_梯度消失与梯度爆炸/)** 的根源
- **二分查找**：每次搜索范围减半，剩余元素数构成公比为 $\dfrac{1}{2}$ 的等比数列

### 2. 等比数列的定义

如果一个数列 $\{a_n\}$ 从第 2 项起，每一项与前一项的比都等于同一个常数 $q$ （ $q \neq 0$ ），即：

$$
\frac{a_{n+1}}{a_n} = q \quad (n = 1, 2, 3, \ldots)
$$

那么 $\{a_n\}$ 就叫做 **等比数列**，常数 $q$ 叫做 **公比（Common Ratio）**。

> **直觉解读**：等差数列像"走楼梯"（每步加 $d$ ），等比数列像"坐电梯"（每步乘 $q$ ）。当 $q > 1$ 时指数增长，当 $0 < q < 1$ 时指数衰减，当 $q < 0$ 时正负交替。

| 公比 $q$ 的范围 | 数列行为 | 例子 |
| -------------- | -------- | ---- |
| $q > 1$ | 指数增长 | $1, 2, 4, 8, 16, \ldots$ |
| $q = 1$ | 恒定 | $3, 3, 3, 3, \ldots$ |
| $0 < q < 1$ | 指数衰减 | $1, 0.5, 0.25, 0.125, \ldots$ |
| $q < 0$ | 正负交替 | $1, -2, 4, -8, \ldots$ |

### 3. 通项公式

从首项 $a_1$ 出发，连续乘以 $(n-1)$ 次公比 $q$ ，就到了第 $n$ 项：

$$
a_n = a_1 \cdot q^{n-1}
$$

> **直觉解读**：这和 **[指数函数](../../03_指数与对数/04_指数函数/04_指数函数.md)** 的形式几乎一模一样！等比数列的通项 $a_n = a_1 \cdot q^{n-1}$ 本质上就是一个以 $q$ 为底的指数函数在正整数点上的取值。**等差数列 ↔ 线性函数，等比数列 ↔ 指数函数**——这个类比非常重要。

### 4. 等比中项

如果 $a$ 、 $b$ 、 $c$ 三个数构成等比数列（且同号），那么中间的 $b$ 叫做 $a$ 和 $c$ 的 **等比中项（Geometric Mean）**。根据等比的定义：

$$
\frac{b}{a} = \frac{c}{b} \implies b^2 = ac \implies b = \pm\sqrt{ac}
$$

> **直觉解读**：等比中项是两端数值的**几何平均值**。在 AI 中，几何平均数常用于评估模型的综合表现——比如困惑度（Perplexity）就是几何平均概率的倒数。

### 5. 求和公式

等比数列的求和需要一个巧妙的技巧——"错位相减法"。设前 $n$ 项和为：

$$
S_n = a_1 + a_1 q + a_1 q^2 + \cdots + a_1 q^{n-1}
$$

两边同时乘以公比 $q$ ：

$$
q S_n = a_1 q + a_1 q^2 + a_1 q^3 + \cdots + a_1 q^n
$$

两式相减，大量项消掉了：

$$
S_n - q S_n = a_1 - a_1 q^n
$$

$$
(1 - q) S_n = a_1 (1 - q^n)
$$

当 $q \neq 1$ 时，两边除以 $(1-q)$ ：

$$
S_n = \frac{a_1(1 - q^n)}{1 - q} = \frac{a_1(q^n - 1)}{q - 1}
$$

当 $q = 1$ 时，数列每项都等于 $a_1$ ，所以 $S_n = n a_1$ 。

> **直觉解读**：求和公式的两种等价形式只是分子分母同时变号。当 $q > 1$ 时用第二种形式 $\dfrac{a_1(q^n - 1)}{q - 1}$ 更方便（避免负号）。

### 6. 无穷等比级数：当 $n \to \infty$

当 $|q| < 1$ 时，随着 $n$ 增大， $q^n \to 0$ 。此时等比数列的项越来越小，前 $n$ 项和趋近一个有限值：

$$
S_\infty = \frac{a_1}{1 - q} \quad (|q| < 1)
$$

这就是 **无穷等比级数（Infinite Geometric Series）** 的和。

举个具体例子： $1 + \dfrac{1}{2} + \dfrac{1}{4} + \dfrac{1}{8} + \cdots = \dfrac{1}{1 - 0.5} = 2$

> 📌 这个结果直觉上也说得通：想象你从 0 开始向 2 走，每步走剩余距离的一半——你永远走不到 2，但会无限接近它。

在 AI 中，无穷级数的概念出现在 **折扣累积奖励** $G = r_1 + \gamma r_2 + \gamma^2 r_3 + \cdots$ 中——这正是一个公比为 $\gamma$ （折扣因子，通常 $0.9$ ~ $0.99$ ）的等比级数。

## 动手实践

```python
# 文件：code/geometric_sequence.py
# 等比数列的通项公式、求和与无穷级数
# 环境要求：Python 3.10+（仅使用标准库）

def geometric_term(a1: float, q: float, n: int) -> float:
    """等比数列通项公式：a_n = a_1 * q^(n-1)"""
    return a1 * q ** (n - 1)


def geometric_sum(a1: float, q: float, n: int) -> float:
    """等比数列求和公式"""
    if q == 1:
        return n * a1
    return a1 * (1 - q ** n) / (1 - q)


def geometric_infinite_sum(a1: float, q: float) -> float:
    """无穷等比级数求和（|q| < 1）"""
    if abs(q) >= 1:
        raise ValueError("|q| 必须小于 1")
    return a1 / (1 - q)


if __name__ == "__main__":
    # 示例 1：复利增长
    print("=" * 50)
    print("示例 1：1000 元存款，年利率 10%，10 年后")
    a1, q, n = 1000, 1.1, 10
    an = geometric_term(a1, q, n)
    print(f"  第 {n} 年末金额：{an:.2f} 元")

    # 示例 2：学习率指数衰减
    print("\n" + "=" * 50)
    print("示例 2：学习率从 0.01 开始，每 epoch 乘以 0.95")
    lr0, decay = 0.01, 0.95
    for epoch in [1, 10, 50, 100]:
        lr = geometric_term(lr0, decay, epoch)
        print(f"  Epoch {epoch}: lr = {lr:.6f}")

    # 示例 3：无穷等比级数
    print("\n" + "=" * 50)
    print("示例 3：1 + 1/2 + 1/4 + 1/8 + ...")
    s_inf = geometric_infinite_sum(1, 0.5)
    print(f"  理论值（公式）：{s_inf}")
    s_approx = geometric_sum(1, 0.5, 30)
    print(f"  前 30 项近似值：{s_approx:.10f}")

    # 示例 4：强化学习折扣累积奖励
    print("\n" + "=" * 50)
    print("示例 4：折扣累积奖励（每步奖励 1，折扣因子 γ=0.99）")
    gamma = 0.99
    s_inf = geometric_infinite_sum(1, gamma)
    print(f"  理论最大累积奖励：{s_inf:.2f}")
    s_100 = geometric_sum(1, gamma, 100)
    print(f"  前 100 步累积奖励：{s_100:.2f}")
```

- 环境要求：Python 3.10+（仅使用标准库）
- 运行命令：`python code/geometric_sequence.py`

## 典型误区

| 误区 | 正确理解 |
| ---- | -------- |
| 认为公比 $q$ 不能为负数 | 公比可以为负数（如 $1, -2, 4, -8, \ldots$ ），但不能为零 |
| 无穷等比级数任何情况都收敛 | 只有 $\|q\| < 1$ 时无穷级数才收敛。若 $\|q\| \geq 1$ ，各项不趋于 0，级数发散 |
| 混淆等比数列通项与等差数列通项 | 等差用加法 $a_n = a_1 + (n-1)d$ ，等比用乘法 $a_n = a_1 \cdot q^{n-1}$ |
| 求和公式忘记 $q = 1$ 的特殊情况 | 当 $q = 1$ 时分母 $1 - q = 0$ ，公式失效。此时 $S_n = na_1$ |

## 练习题

### 练习 1：识别与计算（难度：⭐）

已知等比数列的首项 $a_1 = 3$ ，公比 $q = 2$ ：

1. 写出前 6 项
2. 求第 10 项 $a_{10}$
3. 求前 10 项和 $S_{10}$

<details>
<summary>💡 提示</summary>

通项公式 $a_n = 3 \times 2^{n-1}$ ，求和公式 $S_n = \dfrac{3(2^n - 1)}{2 - 1} = 3(2^n - 1)$ 。

</details>

<details>
<summary>✅ 参考答案</summary>

1. $3, 6, 12, 24, 48, 96$

2. $a_{10} = 3 \times 2^9 = 3 \times 512 = 1536$

3. $S_{10} = 3(2^{10} - 1) = 3 \times 1023 = 3069$

</details>

### 练习 2：反向求参数（难度：⭐⭐）

已知等比数列中 $a_3 = 12$ ， $a_6 = 96$ 。求公比 $q$ 和首项 $a_1$ 。

<details>
<summary>💡 提示</summary>

利用通项公式列方程，然后用 $\dfrac{a_6}{a_3} = q^3$ 来求 $q$ 。

</details>

<details>
<summary>✅ 参考答案</summary>

$$\dfrac{a_6}{a_3} = \dfrac{a_1 q^5}{a_1 q^2} = q^3 = \dfrac{96}{12} = 8$$

∴ $q = 2$

由 $a_3 = a_1 q^2 = 4a_1 = 12$ ，得 $a_1 = 3$

</details>

### 练习 3：无穷级数（难度：⭐⭐⭐）

在强化学习中，折扣累积奖励 $G = \sum_{k=0}^{\infty} \gamma^k r_{k+1}$ 。如果每步奖励恒为 $r = 1$ ，折扣因子 $\gamma = 0.9$ ：

1. 用无穷等比级数公式计算 $G$
2. 至少需要多少步，前 $n$ 步的累积奖励才能达到理论值的 $95\%$ ？

<details>
<summary>💡 提示</summary>

$G = \dfrac{1}{1-0.9} = 10$ 。第二问令 $S_n \geq 0.95 \times 10 = 9.5$ ，然后用对数求解。

</details>

<details>
<summary>✅ 参考答案</summary>

1. $G = \dfrac{1}{1 - 0.9} = 10$

2. 令 $S_n = \dfrac{1 - 0.9^n}{1 - 0.9} = \dfrac{1 - 0.9^n}{0.1} \geq 9.5$

$$1 - 0.9^n \geq 0.95 \implies 0.9^n \leq 0.05$$

两边取自然对数：

$$n \ln 0.9 \leq \ln 0.05 \implies n \geq \dfrac{\ln 0.05}{\ln 0.9} \approx \dfrac{-3.00}{-0.105} \approx 28.4$$

∴ 至少需要 **29 步**。

</details>
