# 图表与证据

> **所属路径**：`00_高中复习/04_科学思维/04_图表与证据`
> **包含知识点**：4 个

---

## 概述

在科学思维的旅程中，我们已经学会了如何设计实验（变量与控制）、如何从观察提出假设（观察与假设），以及如何区分相关与因果。但无论你的实验设计得多么精巧、假设多么有创意，最终你都需要把发现**用图表呈现出来**，让其他人也能看到你看到的东西，并且评判你的证据是否可信。

"图表与证据"是科学思维的最后一环——也是连接"思考"和"表达"的桥梁。在人工智能领域，这个能力尤为关键：训练一个模型后，你需要用损失曲线来判断模型是否在学习、用混淆矩阵来评估分类效果、用折线图来对比不同方案。如果你不会读图、不会判断证据的可靠性、不会清晰地表达结论，那么再好的模型也无法让别人信服。

本主题将从"如何读懂一张图"开始，到"如何判断证据的强弱"，再到"如何正确表达结论"，最后以"如何制作一张好图"收尾，形成完整的"数据→图表→证据→结论"思维链条。

## 学习建议

1. **学习顺序**：建议按 01 → 02 → 03 → 04 的顺序学习。先学会"读图"（图表解读），再学会"判断"（证据强弱），然后学会"表达"（结论表达），最后学会"制图"（数据可视化基础）。这个顺序遵循"先输入后输出"的认知规律。
2. **时间分配**：每个知识点预计 30–45 分钟，整个主题约 2–3 小时。建议在学习"数据可视化基础"时预留更多时间用于动手编程。
3. **常见困难**：最常见的困难是"一看到图表就觉得数据说了什么，但说不清到底是怎么看出来的"。这是因为我们往往依赖直觉读图，而没有养成系统检查标题、坐标轴、刻度、单位的习惯。建议在学完"图表解读"后，找几张新闻中的图表练练手。
4. **动手实践**：本主题的每个知识点都配有 Python 代码示例。尤其是"数据可视化基础"中的 matplotlib 示例，建议你亲自运行、修改参数、观察变化——这将是你踏入数据科学世界的第一步。
