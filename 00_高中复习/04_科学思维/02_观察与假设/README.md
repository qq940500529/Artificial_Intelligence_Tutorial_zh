# 观察与假设

> **所属路径**：`00_高中复习/04_科学思维/02_观察与假设`
> **包含知识点**：4 个

---

## 概述

科学探究的起点不是公式，不是代码，而是一双善于观察的眼睛和一个敢于猜测的大脑。在上一个主题 **[变量与控制](../01_变量与控制/)** 中，我们学会了分辨实验中的各种变量；但在那之前，还有一个更根本的问题——我们是如何发现"某件事好像和另一件事有关"的？答案就是 **观察与假设**。

"观察"是把日常感受转化为可记录、可复现的数据；"假设"是在观察的基础上，大胆而有条理地提出可检验的猜测。这两个步骤看似简单，却决定了后续一切分析的质量。在人工智能领域，数据采集的规范性直接影响模型质量，而"训练一个模型"本身就是在验证一个关于数据模式的假设。

本主题将带你从最朴素的"记录看到了什么"开始，一步步走到"怎么判断我的猜测靠不靠谱"，为后续学习 **[相关与因果](../03_相关与因果/)** 以及 **[图表与证据](../04_图表与证据/)** 打下坚实基础。

## 学习建议

1. **学习顺序**：请严格按照 01 → 02 → 03 → 04 的顺序学习。现象记录是一切的基础，提出假设依赖于观察，验证思路需要先有假设，假说检验思路则是对验证过程的逻辑升级。
2. **时间分配**：建议总计 2–3 小时。前两个知识点各 30 分钟，后两个知识点各 40–50 分钟。
3. **常见困难**：许多初学者会跳过观察直接"拍脑袋"提假设，或者只寻找支持自己猜测的证据而忽视反面证据。学习时请特别留意"验证思路"中关于"证伪"的讨论。
4. **动手实践**：每个知识点都附带了 Python 代码示例，建议亲手运行并修改参数，感受数据从记录到分析的完整流程。
