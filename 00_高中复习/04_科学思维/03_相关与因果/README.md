# 相关与因果

> **所属路径**：`00_高中复习/04_科学思维/03_相关与因果`
> **包含知识点**：4 个

---

## 概述

"相关与因果"是科学思维中最容易被误解、也最值得深入理解的核心话题。日常生活中，我们经常看到两个事物同时出现，就下意识地认为它们之间有因果关系——"每次我带伞就不下雨"、"冰淇淋卖得越多溺水事故越多"。这种直觉有时对，但很多时候会严重误导我们。

在人工智能领域，理解"相关不等于因果"不仅是哲学讨论，更是一个直接影响模型质量的工程问题。机器学习模型天生就是在数据中寻找相关性的机器——如果训练数据中存在虚假的相关性（伪相关），模型就会"学到"这些虚假的模式，导致在新数据上表现糟糕。识别真正的因果关系、排除混杂因素干扰，是建立可靠 AI 系统的必要思维基础。

本主题将带你从认识相关关系出发，逐步理解因果关系的特殊性，见识伪相关的陷阱，最终通过辛普森悖论深刻体会"数据有时候会骗人"这一事实。

## 学习建议

1. **学习顺序**：建议严格按 01 → 02 → 03 → 04 的顺序学习。相关关系是最基本的概念，因果关系在此基础上区分"真正的原因"，伪相关用真实案例揭示相关关系的陷阱，辛普森悖论则是最深刻也最有趣的反直觉现象。
2. **时间分配**：每个知识点预计 30–50 分钟，整个主题约 2.5–3.5 小时。其中辛普森悖论难度较高，建议预留更多思考时间。
3. **常见困难**：最常见的思维障碍是"我明明看到了数据上的关联，为什么不能说是因果？"这个困惑很正常——本主题的核心目标就是帮你建立"对相关性保持警惕"的思维习惯。建议每学完一节，都停下来想一想：生活中有哪些"看起来相关但未必有因果"的例子？
4. **动手实践**：每个知识点都附有 Python 代码示例，建议亲手运行并尝试修改参数，观察结果的变化。数据模拟是理解这些概念最直观的方式。
