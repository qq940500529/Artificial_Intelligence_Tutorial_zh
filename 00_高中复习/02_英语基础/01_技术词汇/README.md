> **包含知识点**：4 个

---

## 概述

在人工智能的学习旅程中，你会不断遇到英文术语——从数学公式中的 variable（变量）、function（函数），到编程中的 loop（循环）、debug（调试），再到 AI 领域的 neural network（神经网络）、machine learning（机器学习）。如果每次遇到新词都要停下来查词典，学习效率会大打折扣。

本主题的目标是帮助你在正式进入人工智能学习之前，建立一套**核心技术词汇表**。我们将词汇按领域分为四组：数学、编程、人工智能和机器学习，覆盖约 **200 个核心术语**。每组词汇都配有中文释义、发音提示、例句和常用搭配语块，帮助你在真实技术场景中快速识别和理解这些术语。

---

## 教学方法说明

本主题的课程设计基于 **ESP（English for Specific Purposes，专门用途英语）** 教学方法论，结合以下四种核心策略：

| 策略 | 说明 | 在课程中的体现 |
| ---- | ---- | -------------- |
| **语块教学** | 学习词汇的常用搭配和固定短语，而非孤立单词 | 每个词汇表增加"常用语块"列；专设"核心语块与搭配"小节 |
| **语境学习** | 在真实技术文本中遇到和理解词汇 | 每课包含"真实语境阅读"小节，使用真实 AI/编程材料 |
| **词根词缀分析** | 通过词根词缀的规律系统性地扩大词汇量 | 每课包含"词根词缀解码"小节，揭示构词规律 |
| **间隔重复** | 按遗忘曲线安排复习节奏，巩固长期记忆 | 每课包含"记忆策略"章节，给出推荐复习计划 |

## 学习建议

**推荐学习顺序**：按照 01 → 02 → 03 → 04 的顺序学习。数学词汇和编程词汇是基础，AI 词汇和机器学习术语建立在前两者之上。

**四步学习法**（每个知识点建议遵循）：

1. **首次学习**（40–50 分钟）：通读全文，重点关注词汇表和语块搭配
2. **语境巩固**（20 分钟）：完成"真实语境阅读"小节的注释练习
3. **主动产出**（15 分钟）：完成全部练习题，特别是语块填空和阅读理解
4. **间隔复习**：按照"记忆策略"章节的复习计划（第 1、3、7、14、30 天）回顾词汇卡片

**常见困难**：
- **发音障碍**：不必追求标准发音，能在脑中建立"看到拼写 → 联想含义"的映射即可
- **记忆遗忘**：属于正常现象。利用间隔重复策略和双语术语卡片（参见 [总结与记笔记](../04_总结与记笔记/)）可显著提高留存率
- **词汇混淆**：部分近义词容易混淆（如 parameter 和 argument），课程中设有"易混词辨析"专栏
- **从单词到语块的跨越**：仅记住孤立单词不够，还需掌握其常见搭配（如不只记 variable，还要记 define a variable、random variable）
