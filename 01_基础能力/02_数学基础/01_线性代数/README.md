## 概述

线性代数是人工智能的"母语"。神经网络的每一层、注意力机制的每一次计算、嵌入向量的每一次相似度比较，背后都是矩阵与向量的运算。可以毫不夸张地说：**不懂线性代数，就读不懂现代人工智能的论文，更写不出真正高效的模型代码。**

本主题面向已经掌握高中向量基础（特别是 [向量表示与运算](../../../../00_高中复习/01_数学基础/06_向量/01_向量表示与运算/01_向量表示与运算.md)、[向量坐标化](../../../../00_高中复习/01_数学基础/06_向量/03_向量坐标化/03_向量坐标化.md) 和 [线性组合与线性相关初步](../../../../00_高中复习/01_数学基础/06_向量/04_线性组合与线性相关初步/04_线性组合与线性相关初步.md)）的学习者，将向量从"二维平面的箭头"扩展到"高维空间的张量"，把矩阵理解为"线性变换的具体表示"，并最终掌握特征值分解、奇异值分解等 AI 中无处不在的核心工具。

我们的讲解策略是：**先用几何直觉建立画面感，再用代数公式精确表达，最后用 NumPy 代码动手验证**。每一个公式背后，都有一张可以画出来、想得通的图。

## 学习建议

**建议学习顺序**：严格按照 01 → 02 → 03 → 04 → 05 → 06 → 07 顺序学习。每个知识点都建立在前面的基础之上，跳读容易在后续遇到障碍。

**时间分配**（总计约 10 小时）：

- 01 向量与矩阵（80 分钟）：建立从"高中向量"到"高维张量"的扩展观
- 02 线性变换（90 分钟）：理解矩阵乘法的几何含义，这是后面所有内容的"灵魂"
- 03 范数与距离（70 分钟）：掌握不同范数的几何意义和应用场景
- 04 正交化与投影（80 分钟）：理解 Gram-Schmidt 和最小二乘
- 05 特征值与奇异值分解（120 分钟）：本主题的"高峰"，PCA、推荐系统、降维都依赖于此
- 06 矩阵求导直觉（90 分钟）：为反向传播打下严格基础
- 07 矩阵分解应用（70 分钟）：把前面学到的工具串起来，了解工业级数值计算如何使用它们

**常见困难点**：

- **维度感知**：高维向量无法画图，需要训练"用代数公式想象几何"的能力。建议每个新概念先在二维或三维想清楚，再推广。
- **矩阵乘法的"阅读方向"**：矩阵乘法不满足交换律，$AB$ 和 $BA$ 含义截然不同。读公式时要养成"从右往左"看的习惯——先把右边的向量"输入"，再被左边的矩阵"作用"。
- **特征值与奇异值的混淆**：特征值定义在方阵上，奇异值定义在任意矩阵上。前者关心"不变方向"，后者关心"主轴长度"。
- **矩阵求导的转置约定**：分子布局与分母布局会导致结果差一个转置。本课程统一使用 **分母布局（denominator layout）**，与机器学习教材主流惯例一致。
