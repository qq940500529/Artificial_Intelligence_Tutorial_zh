## 概述

编程语言是人工智能从业者最基本的工具——就像画家手中的画笔、厨师手中的刀。在整个 AI 学习旅程中，你需要用代码来处理数据、实现算法、训练模型、部署服务。Python 凭借其简洁的语法、丰富的生态和强大的科学计算库，已经成为 AI 领域的首选编程语言。

本主题从零开始，系统地介绍 Python 编程的基础知识。前 3 个知识点覆盖"写出能运行的程序"所需的核心概念——变量、控制流和函数；接下来学习列表推导与生成器这一 Pythonic 编程特性；然后进入异常处理和装饰器与上下文管理器，为安全的文件操作打下基础；最后通过类型提示让代码更清晰、更安全。

## 学习建议

- **学习顺序**：强烈建议按照 01 → 08 的顺序学习。前 3 个知识点是线性依赖关系（变量→控制流→函数）；04 列表推导与生成器是循环和函数的自然延伸；05–06 异常处理和装饰器/上下文管理器为 07 文件操作提供必要的前置知识；08 类型提示作为代码质量工具放在最后。
- **时间分配**：每个知识点预计 40–60 分钟（含动手实践），整个主题约需 6–8 小时。
- **学习方法**：编程必须"动手"——仅仅阅读课程文本远远不够。请务必在自己的电脑上运行每一段示例代码，并尝试修改它们观察结果变化。
- **常见困难点**：
  - 初学者最容易在"可变与不可变类型"、"作用域"和"异常传播"上感到困惑，遇到时不要跳过，多花时间理解。
  - 列表推导和生成器是 Python 的标志性特性，初次接触可能觉得"语法糖"不好读，但熟练后会大幅提升编码效率。
  - 装饰器涉及函数作为参数传递和闭包的概念，如果觉得难，建议先确保函数（03）掌握扎实。
